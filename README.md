# Jetpack Jumper

---

My code can be found in the A3 directory with A3.cpp containing the main game loop.

Shaders can be found in the A3/Assets directory.

Features Implemented:
   -hierarchical modelling
   
   -texture mapping 
   -normal mapping
   -cube mapping
   -a particle system
   -keyframe animation
   -synchronized sound
   -motion blur
   -collisions
   -UI
